## GameLobby CRUD Application with JWT Authentication
#### Angular, Express, Node.JS, Mongo DB

### How to run application locally
#### By default
- git clone --URL 
- cd back-end
- npm install
- npm run dev
- cd ..
- cd front-end-project
- npm install
- ng serve --open
#### Using heroku
- git clone
- cd front-end-project
- heroku local web
